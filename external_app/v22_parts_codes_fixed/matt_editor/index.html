                    💾 Save Editor - Bl4 Save File API
                

                

                    Decrypt, edit, and encrypt Bl4 save files using the Bl4 Save File API
                


                
                

                    

                        🔧 API Configuration
                    

                    

                        
                        
                        

                            Required for .sav decrypt and for encrypt / overwrite. Not required to open an
                            existing .yaml, .yml, or .txt export. Enter a
                            17-digit Steam ID (starting with 7656119) or a 32-character Epic ID (hex).
                        

                    

                


                
                

                    

                        🔓 Decrypt Save File
                    

                    

                        Upload a .sav file to decrypt it to YAML format (Steam/Epic ID required), or upload a .yaml, .yml, or
                        .txt file directly (read in the browser, no ID). Item serials are automatically decoded so you can
                        edit them immediately.
                        


                        💡 Save file location:
                        
                            C:\Users\%USERNAME%\Documents\My games\Bl4\Saved\SaveGames\%SteamID%\Profiles\client
