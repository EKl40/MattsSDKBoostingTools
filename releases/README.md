# Downloads

Use this folder for the current packaged beta download.

- `MSBT_External_Beta.zip` is the normal user package. It includes the SDK mod, the standalone external EXE, resources, and launcher batch file.
- `latest.json` is the tiny version manifest checked by the external app's Update button/banner.
- Source code is available from GitHub's built-in source ZIP on the main repository page.
- Current MSBT builds require SDK 03 / oak2-mod-manager v0.3: <https://github.com/bl-sdk/oak2-mod-manager/releases/tag/v0.3>

For install instructions, start with the top of the main [README](../README.md).
